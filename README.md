# Agentic Financial Research Assistant

An Agentic AI system that autonomously collects financial news and market data to generate structured company insight reports.

## Features

- Multi-agent workflow using LangGraph
- Financial news ingestion via NewsAPI
- Market data retrieval via Yahoo Finance
- LLM-powered sentiment analysis
- Automated company insight report generation
- Streamlit dashboard for visualization

## Tech Stack

- Python
- FastAPI
- LangGraph
- LLM APIs
- Streamlit
- NewsAPI
- Yahoo Finance

## Project Goal

Automate financial research using AI agents that gather, analyze, and synthesize market intelligence into structured reports.
